Início do projeto com npm e instalação do TypeScript

npm init -y
npm install typescript --save-dev
npx tsc --init

Trocar no tsconfig.json de "module": "nodenext" e "target": "esnext" para "module": "ESNext" e "target": "ES6"

"compilerOptions": {
    "module": "ESNext",
    "target": "ES6",
    "types": [],
    ...
}

Adicionar no package.json dentro de scripts para executar npm run build e npm start

"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "build": "tsc",
    "start": "node src/app.js"
  },

e executar no terminal

npm run build
npm start
