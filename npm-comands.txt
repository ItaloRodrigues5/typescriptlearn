Início do projeto com npm e instalação do TypeScript

npm init -y
npm install typescript --save-dev
npx tsc --init

Trocar de "module": "nodenext" e "target": "esnext" para "module": "ESNext" e "target": "ES6"

"compilerOptions": {
    "module": "ESNext",
    "target": "ES6",
    "types": [],
    ...
}

Adicionar no package.json para executar npm run build e npm start

"build": "tsc",
"start": "node src/app.js"

e executar no terminal

npm run build
npm start
